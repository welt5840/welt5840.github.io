Introduction

HTML stands for HyperText Markup Language and it is the standard markup language for documents designed to be displayed in web browsers. Markup languages refer to computer languages that consist of keywords, names, or tags to control its structure and relationships.

Within the HTML language, there is another used to describe the presentation of a document. Cascading Style Sheets (CSS) is used to style and layout web pages. For example:

  • Altering fonts, size, color, and spacing of content.
HTML Basics

HTML consists of a series of elements that are used to describe the structure of a web page. Elements within this language tell the browser how to display its content.

Here is some basic knowledge one should know for a simple HTML document:

  • A general understanding of the Internet and the World Wide Web (WWW).
  • Good working knowledge of HyperText Markup Language (HTML).
Starting HTML
To get started with writing an HTML, open your choice of text editor and write: The <!DOCTYPE> Declaration
The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly.
It must only appear once, at the top of the page (before any HTML tags).
The <!DOCTYPE> declaration is not case sensitive.
The <!DOCTYPE> declaration for HTML5 is:
<!DOCTYPE html>
Elements

An HTML element is defined by a start tag, some content, and an end tag.
The HTML element is everything from the start tag to the end tag: <tagname>Content goes here...</tagname>

This element code above is:
<tagname>Content goes here...</tagname>

Examples of some HTML elements:

My First Heading

This element code above is:
<tagname> My First Heading</tagname>

My first paragraph.

This element code above is:
<tagname>tagname > My first paragraph</tagname>

Attributes

Another important aspect of HTMLs are attributes.

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

The href Attribute The tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to: Example < a href="https://www.w3schools.com">Visit W3Schools You will learn more about links in our HTML Links chapter. The src Attribute The tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed: Example < img src="img_girl.jpg">
